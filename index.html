怎么用

  
    
    
  1. 填档案：到「亲人档案」把 TA 的口头禅、性格、你们的回忆填进去，保存。
    2. 
    
  2. 选声音：到「声音」挑一个最接近 TA 的音色，调语速音调。
    3. 
    
  3. 接大脑（可选但强烈推荐）：到「大脑」填一个大模型 key，TA 会聪明得多。
    4. 
    
  4. 开聊：回到「聊天」，点 🎤 说话，或直接打字。
    5. 
  

  
常见问题

  
    
    
  • 点了 🎤 没反应 / 识别不出来：手机上 iPhone 用 Safari（注意：加到主屏幕的图标里系统不提供语音识别，请回 Safari 用）、安卓用 Chrome；电脑上推荐 Edge。都要先允许麦克风权限。
    • 
    
  • 没有声音：检查「声音」里引擎不是"静音"；系统音量；试听按钮。
    • 
    
  • 大模型报错：多数是 key 不对或该服务商不允许浏览器直连（CORS），换 DeepSeek 或 Claude 试试。
    • 
    
  • 数据存在哪：档案、key、聊天记录全部只存在本机浏览器 localStorage，不会上传。换浏览器/清缓存会丢，记得用「导出备份」。
    • 
  

  
声音克隆（进阶）

  
    
    
  • 准备亲人 1~3 分钟清晰、无背景音的说话录音（微信语音导出也行）。
    • 
    
  • 方案 A：ElevenLabs 官网做 Instant Voice Clone，把 Voice ID + API Key 填到「声音」页。
    • 
    
  • 方案 B（免费/本地）：自建 GPT-SoVITS 或 CosyVoice，把服务地址填到"自定义 TTS 接口"。
    • 
  

  
一点提醒

  
克隆真人的声音和人格请先征得本人同意；如果是为了怀念已离开的亲人，请留意自己的情绪，这个工具是陪伴，不是替代。
